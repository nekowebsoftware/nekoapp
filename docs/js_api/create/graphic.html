nekoapp.create.graphic()

This function is used to create application vector graphic objects like icons, logos, etc. If your application has graphic library then you can use graphics from there.

var your_graphic = nekoapp.create.graphic(your_app, graphic, viewBox, options);

Function parameters:

  • First parameter determines to which application is object about to be linked. That means function receives only nekoapp objects and also it means that it supposed to be your application if you want this object added to your application.
  • Second parameter determines your graphic. It can be graphic name, so Neko Web Application will insert it from graphics library (if it exists). You just need to set graphic set name where you needed graphic is stored and its name:graphicset_name;graphic_nameor it can svg inside contents as object or string.
  • Third parameter determines SVG's view box. It must be string with 4 numbers: x start position, y start position, width and height.
  • Last parameter is optional and determines what attributes will be included in your object. It's a object with properties.
    Here's the list of options:
    • id - sets id attribute with id value of your object.
    • class - sets class attribute with class value.
    • attr - sets custom attributes of your element object. It's object where you define attributes with its values.
    • style - sets style attributes with style properties. It can be string or object with properties and its values.

Function returns:

Application graphic object that you can put wherever you want.

Function example:

var your_graphic = nekoapp.create.graphic(your_app, "main;logo_head", "0 0 180 18");

your_app.modules.your_app_module.appendChild(your_graphic);

Function since:

The beginning of Neko Web Application's life
