Setting App

So, Neko Web Application is a JavaScript framework. Why? The main goal is to dynamically load and create page parts to optimize loading and make work responsive. So, you need to write main application script that does all application work.

So let's start with creating script with name like

your_app.main.js

and then define your app

var your_app = new nekoapp();

nekoapp counstructor takes one argument as object with all required properties for application.

Now we need to set these properties.

Application Info

First, application info is a block that have application info such like title, version, url and most important nekoappID as unique identifier for your application.

var your_app = new nekoapp({
     applicationInfo : {
          nekoappID : "21" ,
          applicationTitle : "Your Application" ,
          applicationVersion : "1.0.0" ,
          applicationURL : "//www.example.com/"
     }
});

To get own nekoappID you need to register your appplication in Application Manager.

Or you can use 0 for trying or testing out framework and/or your application.

Registering Application

So, you need to register your application. First, you need to use your account for this.

If you have, click "Log in" button, if not click "Create account" and register your account.

If you done with previous step and in dashboard page, do next:

  1. Click "New app" button.
  2. In appeared window set your app credentials.
  3. Click "Add" button to register your app.

When you done registering, in dashboard table right of title in second column is your application ID, set it in "nekoappID" property.

Example:

Application Title nekoappID Version URL
Your Application 21 1.0.0 //www.example.com/

The next step is your application preferences.
